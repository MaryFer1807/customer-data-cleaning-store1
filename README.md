## Project Overview
Store 1, an e-commerce company, is preparing the launch of a Customer Loyalty Program. To support this initiative, the company needs clean, structured, and reliable customer data to design personalized marketing campaigns and analyze customer behavior.

This project focuses on data cleaning and preparation as the foundation for future analysis and segmentation.

## Objective
Prepare and transform customer data to:
- Ensure data quality and consistency.
- Calculate key business metrics (KPIs).
- Enable future customer segmentation and behavioral analysis.

## Tasks Performed
- Cleaned customer profile data.
- Standardized customer names.
- Validated and corrected age values.
- Calculated total spending per customer.
- Verified consistency between product categories and spending amounts.
- Prepared data for further analysis and visualization.

## Dataset Description
The data was provided as a Python list with the following fields:
- `usuario_id`: Unique customer identifier.
- `usuario_nombre`: Customer name.
- `usuario_edad`: Customer age.
- `categorias_fav_low`: List of purchased product categories.
- `gasto_por_categoria`: List of spending amounts per category.

## Tools Used
- Python
- Jupyter Notebook
- Data cleaning and validation techniques

## Outcome
The result is a clean and structured dataset ready for:
- Customer segmentation
- Purchase behavior analysis
- KPI generation
- Personalized marketing strategies

This project represents the first stage of a multi-step analysis, with deeper insights developed in the following sprint.
